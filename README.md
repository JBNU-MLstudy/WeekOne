map과 비슷한 문법  <br>
List Comprehension <br>
: 리스트 안에 식(for, if 조건문 등)을 지정하여 리스트를 생성하는 것 <br><br>

예) <br>
a = 0 <br>
for a in range(10): <br> 
   a = a+1 <br>
   print(a) <br>
<br>
-> list comprehension   <br> 
b = [x for x in range(10)] # 머신러닝을 위한 파이썬에서 나옵니다. <br>
print(b) <br>
<br>
map: 반복가능한 객체를 받아서 각 요소에 함수를 적용해주는 함수 <br> 
<br>
